# システム概要と設計思想

## 概要

このプロジェクトは、ChatGPT Custom GPT と WordPress REST API を連携させた、個人ブログ向けの記事自動投稿システムです。

音声入力やチャット形式でGPTに指示を出すだけで、記事の生成からWordPressへの下書き保存までを自動化します。

## 設計思想

### 1. シンプルな操作フロー

技術的な複雑さをGPTの裏側に隠し、ユーザーは「記事の内容を話すだけ」で完結するインターフェースを目指しました。

### 2. 安全な自動化

AIが生成した記事を即座に公開するのではなく、必ず「下書き」として保存します。
人間が確認してから公開するという二段階構成により、誤投稿を防ぎます。

### 3. 記事スタイルの分岐

LivetubeSTARブログには2種類の記事スタイルがあります：

- **AI活用記事**: AIとのやり取り・試行錯誤・過程を重視した記録
- **通常体験記事**: グルメ・レビュー・お出かけなどの体験ベースの記録

GPTが記事の内容を判定し、それぞれに最適なテンプレートで構成します。

### 4. セキュリティファースト

- APIキーはWordPressの `wp-config.php` で管理
- GitHubにはキーを一切コミットしない
- ChatGPT側のAPIキーはシークレットとして登録

|------|-----------|------|
| ChatGPT | GPT-4o | 記事生成・アクション実行 |
| OpenAPI | 3.1.0 | GPTアクション定義 |
| WordPress | 6.x | ブログCMS |
| PHP | 8.x | カスタムプラグイン |
| WordPress REST API | v2 | 記事投稿エンドポイント |
